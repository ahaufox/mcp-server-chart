# 功能开关清理工作流 (Feature Flag Cleanup)

## 执行步骤

### 1. 识别过时开关
- **动作**: 查看功能管理后台或配置文件，找出已经上线超过 1 个月的 "Always On" 开关。

### 2. 移除代码路径
- **动作**: 搜索代码库中该 Flag 的判断逻辑。
- **删除**: 移除 `if feature_flag: ... else: ...` 中的 `else` 分支（假设功能已永久开启），并删除 Flag 的配置定义。

### 3. 清理配置环境
- **动作**: 从 `.env.example` 和生产环境变量中移除该 Flag 的 Key。

### 4. 验证回归
- **动作**: 运行测试套件，确保移除 Flag 逻辑后不会导致意外的功能回退。
